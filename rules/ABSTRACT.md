## 通用规则结构

### 分形文档三层结构（通用）

```
项目根目录
├── [文档文件名]           ← 第1层：全局地图
├── [模块目录]/
│   ├── [文档文件名]      ← 第2层：模块清单
│   └── [源文件]          ← 第3层：文件头
└── rules/
    └── [规范文件]         ← 开发规范
```

### 强制同构流程（通用）

```
代码变更
    ↓
更新第3层（文件头 History）
    ↓
更新第2层（如成员/接口/依赖变化）
    ↓
更新第1层（如架构/模块变化）
    ↓
构建验证
    ↓
自检同构
```

### 回环检查清单（通用）

- 第3层：文件头 History 已更新
- 第3层：文件头 Description 与代码一致
- 第2层：组件文档成员清单完整
- 第2层：接口契约与 export 一致
- 第2层：依赖关系准确
- 第1层：模块结构表完整
